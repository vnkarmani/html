Nested tags

one tag can contain another tag is called nested tag.

example of nested tag:

<!DOCTYPE html>

<html>

<head>

<title>We can write here Page or document Title</title>

</head>

<body>

<h1>We can write here Heading through <h1> to <h6></h1>

<p>We can write here any paragraph.</p>

</body>

</html>

-Here,The <html> element is the root element and it defines the whole HTML document.

-It has a start tag <html> and an end tag </html>.

-Then, inside the <html> element there is a <head> & <body> tags:

-so,we can say that here <head> tag and <body> tag are nested tag
