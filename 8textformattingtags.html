Text formatting tags

  1. <b>

    2. This is a tag, which is used to bold the text written between it.

  2. <strong>

    3. This is a tag, which tells the browser that the text is important,its also used to make text bold.

  3. <i>

    4. This is a tag which is used to make text italic.

  4. <em>

    5. This is a tag which is used to display content in italic.

  5. <mark>

    6. This tag is used to highlight text.

  6. <u>

    7. This tag is used to underline text written between it.

  7. <tt>

    8. This tag is used to appear a text in teletype. (not supported in HTML5)

  8. <strike>

    9. This tag is used to draw a strikethrough on a section of text. (Not supported in HTML5)

  9. <sup>

    10. It displays the content slightly above the normal line.

  10. <sub>

    11. It displays the content slightly below the normal line.

  11. <del>

    12. This tag is used to display the deleted content.

  12. <ins>

    13. This tag displays the content which is added

  13. <big>

    14. This tag is used to increase the font size by one conventional unit

  14. <small

    15. This tag is used to decrease the font size by one unit from base font size.
