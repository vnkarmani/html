Website
A website is a collection of many web pages, and web pages are digital files that are written using HTML(HyperText Markup Language).
WWW
The World Wide Web is also known as the web, WWW or W3 refers to all the public websites or pages that users can access on their local computers and other devices through the internet.
Client server architecture
Client server architecture refers to a system that manages resources and services that the client requests. client=our browser
Domain
A domain is a network of computers and devices that are controlled by one set authority and have specific guidelines.it represents an IP address,an URL to find a website on the internet.
Hosting
Hosting is an online service that makes your websites content accessible on the internet.
Address bar
A text box in a web browser displaying the address of the web page
Title bar
The title bar is a horizontal bar located at the top of a window, It displays the title of the software, name of the document or file.
Extension
Extension indicates the file formate or file type, EXAMPLE: .PDF, .TXT, .JPG, .MP4
File type
A file type describes the structure or format of a file
URL/URI
A URL,stands for a UNIFORM RESOURCE LOCATOR is a web address pointing to a specific website, a web page, or a document on the internet. A URI,stands for a UNIFORM RESOURCE IDENTIFIER a string of characters used to identify a resource on a computer network, of which the best known type is the web address or URL.
SEO
SEO stands for “search engine optimization.” In simple terms, SEO means the process of improving your website to increase its ranking in Google,or other search engines.
