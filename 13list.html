Html list

There are three types of list in html.

1.ordered list<ol>

2.unordered list<ul>

3.description list<dl>

1.ordered list <ol>

this tag defines an ordered list. An ordered list can be numerical or alphabetical.

the <li> tag is used to define each list item.

"reverse" attribute is used to specifies the list order should be reverse.

"start" attribute is used to specify start value of an ordered list.

"type" attrinute is used to specify the kind of marker to use in the list.

ex:

<ol>

<li>list item</li>

<li>list item</li>

</ol>

2.unordered list <ul>

this tag defines an ordered list.

the <li> tag is used to define each list item.

"type" attribute is used to specifies that which kind of marker used in the list.

ex:

<ul>

<li>list item</li>

<li>list item</li>

</ul>

3.description list <dl>

The <dl> tag defines a description list.

The <dl> tag is used with <dt> and <dd> tag.

where,The <dt> tag defines a term/name in a description list.

And the <dd> tag describes each term/name.

Inside a <dd> tag we can put paragraphs, line breaks, images, links, lists, etc.

these all are display block elements.

ex:

<dl>

<dt>term name </dt>

<dd>describe term name </dd>

</dl>
