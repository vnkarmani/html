Tags in Html:

  1. <a>

    2. it is used to create hyperlink from one page to another.

    the most important attribute of <a> tag is "href" attribute

    A linked page is normally displayed in the current browser window, unless you specify another target.

    Ex:

    <a href="url link of google" target="_blank">google</a>

  2. <address>

    3. this tag defines the contact information for the author/owner of a document or an article.

    The text in the <address> element usually dispaly in italic, and browsers will always add a line break before and after the this element.

    it is a display block level elemant.

    Ex:

    <address>----</address>

  3. <article>

    4. it is used to crate any article,or we can write any article like news or any blog

    Ex:

    <address>----</address>

  4. <blockquote>

    5. this tag specifies a section that is quoted from another source.

    Ex:

    <blockquote>-----</blockquote>

  5. <br>

    6. this tag inserts a single line break.

    this tag is an empty tag which means that it has no end tag.

  6. <cite>

    7. this tag defines the title of a creative work.

    The text in the this element usually renders in italic.

  7. <code>

    8. Define some text as computer code in a document:

    Ex:

    The HTML <code>button</code> tag defines a clickable button.

  8. <data>

    9. this tag is used to add a machine-readable translation of a given content.

    This element provides both a machine-readable value for data processors, and a human-readable value for rendering in a browser.

    If the content is time- or date-related, use the <time> element instead.

  9. <details> and <summary>

    10. the <details> Specify additionsl details that the user can open and close on demand:

    The <details> tag is often used to create an interactive widget that the user can open and close. By default, the widget is closed. When open, it expands, and displays the content within.

    Any sort of content can be put inside the <details>tag.

    The <summary> tag is used in conjunction with <details> to specify a visible heading for the details.

    Ex:

    <details> <summary>visible heading</summary> </details>

  10. <footer>

    11. this tag defines a footer for a document or section.

    this element contains authorship information,copyright information,contact information any of this.

    You can have more than one <footer> elements in one document.

    Contact information inside a <footer> element should go inside an <address> tag.

    it is a display block level element.

  11. <h1> to <h6>

    12. The <h1> to <h6> tags are used to define HTML headings.

    <h1> defines the most important heading. <h6> defines the least important heading.

    Only use one <h1> per page - this should represent the main heading/subject for the whole page. Also, do not skip heading levels - start with <h1>, then use <h2>, and so on.

  12. <q>

    13. this tag defines a short quotation.

    Browsers normally insert quotation marks around the quotation.

    Use <blockquote> for long quotations.

    Ex:

    <blockquote> <q>----</q> </blockquote>

  13. <div>

    14. this tag defines a division or a section in an HTML document.

    this tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

    this tag is easily styled by using the "class" or "id" attribute.

    Any type of content can be put inside the <div> tag

    By default, browsers always place a line break before and after the <div> element.

    Ex:

    <div class="" id="">---</div>

  14. <span>

    15. this tag is an inline container used to mark up a part of document.

    this tag is easily styled by css or manipulate by java script,using class ir id attribute.

    this tag is same as a <div> tag,but here <span> tag is an inline element so that,browsers can not place a line break before and after the <span> element.when <div> is display block element.

    ex:

    <span class="" id="">---</span>

  15. <section>

    16. this tag is defines a section in document.

    same as,<div> tag this is also display block element.

    ex:

    <section>----</section>
