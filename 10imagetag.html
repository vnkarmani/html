Image tag

image tag is used to merge any image in html document.

images are linked to web pages.

image tag has required two attributes majorly

src: Specifies the path to the image

alt:alternavit text,in any case if image cant load on web page.

we can also specify height and width of an image,where both are given in pixels.

<img src="" alt="" heigt="" width="">

we can also link an image to another document, simply use the <img> tag inside an <a> tag

image map:

here two elements are used for image map,1<area> element and 2<map>.

The <area> tag defines an area inside an image map,which is an image with clickable areas.

<area> elements are always used inside a <map> tag.

"usemap" attribute is used in <img> tag which is associated with <map> elements "name" atttribute,and create relationship between image and map.

ex:

<img src"" alt="" height="" width="" usemap="#vishal">

<map name="vishal">

<area shape="rect" coords="" href="" alt="image not found">

as above example "shape" is an attribute of area tag,it is specifies shape of the area.

"coords" attribute is specifies the coordinates of the area.
