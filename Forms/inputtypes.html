Input types

different input types as shown below:

<input type="">

1.type="text"

it defines a single-line text input field.

2.type="password"

it defines a password field.

The characters in a password field are masked.

3.type="submit"

it defines a button for submitting form data to server file.

the server file is specified in the form's "action" attribute.

If you dont specify submit button's "value" attribute, the button will get a default text.

4.type="reset"

it defines a reset button that will reset all form values to their default values.

If you change the input values and then click the "Reset" button, the form-data will be reset to the default values.

5.type="radio"

it defines a radio button.

Radio buttons let a user select ONLY ONE of a limited number of choices.

6.type="checkbox"

it defines a checkbox.

Checkboxes let a user select ZERO or MORE options of a limited number of choices.

7.type="button"

it defines a button.

8.type="color"

it is used for input fields that should contain a color.

9.type="date"

it is used for input fields that should contain a date.

we can also use the "min" and "max" attributes to add restrictions to dates.

10.type="datetime-local"

it specifies a date and time input field, with no time zone.

11.type="email"

it is used for input fields that should contain an e-mail address.

12.type="image"

it defines an image as a submit button.

The path to the image is specified in the "src" attribute.

13.type="file"

it defines a file-select field and a "Browse" button for file uploads.

14.type="hidden"

it defines a hidden input field which is not visible to a user.

A hidden field lets web developers include data that cannot be seen or modified by users when a form is submitted.

15.type="month"

it allows the user to select a month and year.

16.type="number"

it defines a numeric input field.

we can also set restrictions on what numbers are accepted.

17.type="range"

it defines a control for entering a number whose exact value is not important.

its Default range is 0 to 100. However,we can set restrictions on what numbers are accepted with the "min","max", and "step" attributes.

18.type="search"

it is used for search fields,it behave like a regular text field.

19.type="tel"

it is used for input fields that should contain a telephone number.

20.type="time"

it allows the user to select a time with no time zone.

a time picker can show up in the input field.

21.type="url"

it is used for input fields that should contain a URL address.

the url field can be automatically validated when submitted.

22.type="week"

it allows the user to select a week and year.

a date picker can show up in the input field
